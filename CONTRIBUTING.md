# Contributing to Awesome Vibe Computing

Thanks for your interest in contributing!

## Guidelines

### Adding a new entry

- **One item per pull request** - makes review easier
- **Format**: `[Name](https://url) - Description.`
- **Description**: One sentence, ending with a period. Explain why it matters for vibe computing.
- **No duplicates**: Search existing entries first
- **Quality over quantity**: Only add things that are genuinely useful, not just complete

### What belongs here

This list is about **vibe computing infrastructure** — the tools, protocols, and systems that let AI agents manage your digital and physical life.

✅ Good fit:
- Tools that connect AI to real-world systems
- Infrastructure for running persistent agents
- Voice, home automation, health integrations
- Developer tools for building agentic systems

❌ Not a good fit:
- Generic AI chatbots without agent capabilities
- Browser-based AI website builders
- Tools that don't involve autonomous AI

### Pull request process

1. Fork the repo
2. Add your entry in alphabetical order within its section
3. Submit a PR with a clear title
4. Wait for review

### Questions?

Open an issue if you're unsure whether something belongs.
